# Code of Conduct

## Our Standard

GlialNode aims to be a respectful, constructive project.

Examples of behavior that help create a positive environment:

- assuming good intent
- giving specific, actionable feedback
- welcoming questions from contributors at different experience levels
- discussing disagreements with curiosity rather than contempt

Examples of unacceptable behavior:

- personal attacks
- harassment or intimidation
- dismissive or hostile language
- publishing private information without consent

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, and discussions tied to the repository.

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainer privately when possible.

## Enforcement

Project maintainers may remove comments, close threads, or block participation when necessary to protect the community and keep collaboration productive.
